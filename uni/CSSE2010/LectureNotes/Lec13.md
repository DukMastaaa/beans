# Lecture 13

`LD`, `LDS`, `ST`, `STS`

Data is usually copied, not moved. 

- Register -> register: MOV
- Memory -> register: LD
- Register -> memory: ST
- I/O register -> register: IN
- Register -> I/O register: OUT

Dyadic - 2 operands, monadic - 1 operand.

`SET`, `CLR`, `LSL`, `LSR`, `ASR`, `ROL`, `ROR`
